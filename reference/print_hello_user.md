# Print hello user

This function just prints hello user.

## Usage

``` r
print_hello_user(x)
```

## Arguments

- x:

  This argument does nothing. If it did do something, it might require
  more words to explain.

## Value

Prints `"Hello, User"` to the console

## Examples

``` r
print_hello_user(x = 1)
#> [1] "Hello, user!"
```
