# Print hello world

This function just prints hello world.

## Usage

``` r
print_hello_world(x)
```

## Arguments

- x:

  This argument does nothing. If it did do something, it might require
  more words to explain.

## Value

Prints `"Hello, world!"` to the console

## Examples

``` r
print_hello_world(x = 1)
#> [1] "Hello, world!"
```
