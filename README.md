# metaboTYPE

<!-- badges: start -->

<!-- badges: end -->

The goal of metaboTYPE is to allow users to align multiple runs of indirect calorimetry data with different starting points and quickly analyze and visualize the data for further analysis. 

## Installation

You can install the development version of metaboTYPE2.0 from
[GitHub](https://github.com/) with:

``` r
# install.packages("pak")
pak::pak("Meagan-Kingren/metaboTYPE2.0")
```
